# First, see how your mailboxes are distributed
.\Get-MailboxesOverThreshold.ps1 -ShowDistribution

# Then run in batches
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "A-D"
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "E-H"
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "I-L"
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "M-P"
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "Q-T"
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "U-Z"

# Single letter if needed
.\Get-MailboxesOverThreshold.ps1 -ThresholdPercent 75 -LetterRange "S"
```

**The `-ShowDistribution` flag** gives you a breakdown like:
```
Letter  Count  Percentage
------  -----  ----------
A       412    5.2%
B       687    8.6%
C       523    6.5%
...
